# The Little Writing Robot

Once upon a time, Notra had a little writing robot.

The robot looked at a box of work and sometimes said, "Hmm, this box has a different name, so maybe I should not write about it."

We taught the robot a new rule:

"The name on the box does not matter. If someone picked this box, use the brand voice and write from what is inside."

Then we gave the robot a stronger pencil.

Before, many jobs used the small Haiku pencil by default. Now create-content, schedules, events, brand analysis, routing, and on-demand generation use the Sonnet 4.6 pencil instead.

But we left the Haiku pencil on the shelf in chat.

If a user walks over and chooses it themselves, the robot can still use it.
